Blackcoin More

What is Blackcoin?

Blackcoin is a decentralised digital currency with near-instant transaction speeds and negligible transaction fees built upon Proof of Stake 3.0 (PoSV3, BPoS) as introduced by the Blackcoin development team.

What is Blackcoin More?

Blackcoin More is the name of open source software which enables the use of this currency. It takes Blackcoin to the next level by building upon Bitcoin Core 0.13.2 with some patches from newer Bitcoin Core versions to offer performance enhancements, wider compatibility with third party services and a more advanced base.

How to upgrade?

If you are running Blackcoin Legacy wallet v1.2.x:

You need to download the blockchain and send your funds to the new wallet.

If you are running Blackcoin Lore wallet v2.12.x:

Technically you can point Blackcoin More installation to your data folder, but it is recommended to create a new installation in the default data folder. The same for your funds, it is technically possible to use the same wallet.dat file, but it is better to use the wallet created by new installation. Send your funds to this new wallet.

If you are running Blackcoin More wallet v2.13.x:

You can upgrade just by replacing binaries.
